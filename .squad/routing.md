# Routing Rules

| Signal Pattern | Route To | Notes |
|---------------|----------|-------|
| Architecture, scene hierarchy, system design, scope | Lead | Design and decomposition |
| GDScript, scenes, gameplay, UI, input, nodes | Builder | Implementation |
| Validation, scenarios, harnesses, test failures | Validator | Quality & test automation |
| "Team", multi-domain | Lead + relevant agents | Fan-out |
