## Full Game Loop

1. Level loads → INITIAL_PLANNING phase
2. Player plans defenses per lane
3. Player selects "Start Battle" → BATTLE_PLANNING phase
4. Player deploys units via queue → BATTLE phase
5. Units move through lanes, defenses attack
6. Win condition (unit reaches end) or Lose condition (all units dead)
7. Ready for next level or game end
